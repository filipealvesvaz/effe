HTML:

<h1>

You can use this tag to add headings. Change the number (h1, h2, ...,h6) to make some organizational headings and sub-headings.

<p>

You can use this tag to add a paragraph that describes the heading.

<b>

You can use this tag to apply font weight to any text in the paragraph.

<em>

You can use this tag to emphasize any text in the paragraphs.

<div>

You can use this tag to boxifying the page design. To be able to see modifications you need to use CSS. It is not cover in this project to keep simplicity.

<span>

You can use this tag to group inline-elements and then apply a style like blue color for example.

CSS:

font-family

The font-family property specifies the font for an element.

Aligning Text Horizontally

specifies the horizontal alignment of text in an element.

cascade

The final appearance of a web page is a result of different styling rules.

descendant selectors

These selectors are used to select elements that are descendants of another element.

box model;

All HTML elements can be considered as boxes. The CSS box model represents the design and layout of the site.

tables

The look of an HTML table can be greatly improved with CSS
